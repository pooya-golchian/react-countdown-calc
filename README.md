# React Countdown Calculator

React countdown calculator built with creact-react-app (react 16).
Countdown to Christmas 2018 and anytimes you want :-)



#### Demo

Live Demo URL [LIVE DEMO][1]


#### Install

Install `yarn install`

#### Build and Deploy

Build `yarn build`
Deploy `yarn deploy`
